## Projects

### [Snake Game](./snake-game)
A classic Snake game built with vanilla HTML, CSS, and JavaScript. Features include:
- Three speed levels (Slow, Medium, Fast)
- Pause/Resume functionality
- Score tracking with high score persistence
- Responsive design

[Play the game](./snake-game/index.html) | [View Documentation](./snake-game/README.md)

### [Movie & TV Show Recommendations (2022-2025)](./movie-tv-recommendations-2022-2025/)
A comprehensive research project featuring top 10 recommendations across all major genres for both movies and TV shows from 2022-2025. Each entry includes IMDB ratings, Rotten Tomatoes scores, detailed overviews, and both critical and audience reviews.
